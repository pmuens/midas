# Midas

Automated Cryptocurrency Hedge Fund.

## Useful Commands

```sh
# Run main script
npx ts-node src/main.ts

# Build
npm run build

# Watch
npm run watch

# Compile Smart Contracts
npx hardhat compile

# Run a local Hardhat Network node
npx hardhat node

# Run Hardhat scripts
npx hardhat run --network <network> scripts/<script>.ts
```
